# Eye Tracking & 3D FOV Simulation

## 🎮 Как использовать

1. **Запуск программы**
   Запустите главный файл интерфейса через терминал или вашу IDE:
   ```bash
   python gui_main.py
   ```
2. **Включение камеры**
   В левом меню интерфейса нажмите **«Включить камеру»**. Убедитесь, что ваше лицо хорошо освещено и попадает в кадр на вкладке «Видеопоток».
3. **Калибровка (Обязательно)**
   Сядьте ровно, посмотрите строго в центр вашего монитора и нажмите кнопку **«Сброс и Калибровка»**. Программа запомнит это положение как центральную (нулевую) точку координат.
4. **Запуск 3D Симуляции**
   Нажмите оранжевую кнопку **«Запустить 3D Сцену»**. В отдельном окне откроется 3D-движок. Теперь 50-градусный конус обзора будет плавно следить за перемещением вашей головы и глаз.
5. **Запись данных (Опционально)**
   Чтобы собрать метрики для аналитики, нажмите «Выбрать папку сохранения», а затем **«НАЧАТЬ ЗАПИСЬ»**. Координаты глаз и головы будут сохраняться в `.csv` файл.

---

## 📁 За что отвечает каждый файл

* **`gui_main.py`**
  Главный входной файл. Управляет графическим интерфейсом (CustomTkinter), отрисовкой 2D-радаров (Matplotlib) и фоновой отправкой координат по локальной сети.
* **`visualizer_3d.py`**
  Автономный 3D-движок (Ursina). Работает в параллельном процессе, ловит UDP-пакеты с координатами от `gui_main.py` и рендерит физиологически точный конус обзора в пространстве.
* **`face_tracker.py`**
  Модуль компьютерного зрения. Обертка над нейросетью MediaPipe. Находит в кадре лицо и извлекает координаты ключевых точек (зрачки, края радужки, контуры век, кончик носа).
* **`coordinate_calculator.py`**
  Математическое ядро. Высчитывает отклонения головы от центра, нормализует взгляд в процентах относительно ширины глаза и применяет алгоритмы сглаживания (EMA) для устранения дрожания.
* **`camera_handler.py`**
  Инкапсулированная логика работы с оборудованием. Отвечает за стабильный захват кадров с веб-камеры через OpenCV и безопасное освобождение ресурсов при выходе.
* **`main.py`**
  Запасной (отладочный) скрипт. Минималистичная версия трекера без сложного графического интерфейса и 3D-движка. Используется для тестирования работы камеры и алгоритмов распознавания.
